# nosbin
The decentralized pasting platform, built on nostr

### Features
- [X] Markdown Support
- [X] Use anonymous pubkey
- [X] Use own pubkey
  - [ ] bech32 support
  - [X] nox2x support
- [ ] Preview before posting
- [ ] Comments
- [ ] Reactions
- [ ] Profiles
- [ ] Search
- [ ] Posting to multiple Relays
- [ ] Lightning Tips
